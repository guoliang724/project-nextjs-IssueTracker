### Issue Tracker

This is a fast, responsive application.

#### Please visit: https://issue-tracker-bay-alpha.vercel.app/ . Thanks for visiting

#### Stack:

Next.js14 + Typescript + Tailwind + Prisma + Mysql

#### Deploy:

Database: Google Cloud Mysql  
Server: Vercel

#### Features:

##### Core:

- Creating an issue
- Viewing issues
- Updating an issue
- Deleting an issue

##### Advanced:

- User authentication
- Assigning issues
- Sorting issues
- Filtering issues
